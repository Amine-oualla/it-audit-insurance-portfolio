# It-audit-insurance-portfolio

Projet personnel : audit IT, risques cyber, data analytics et IA dans le secteur assurantiel.

## Portfolio – Audit IT & Data Analytics (Secteur Assurances)

Ce dépôt présente un projet complet d’audit IT réalisé dans le cadre d’un portfolio personnel.  
Il simule une mission d’audit interne dans une compagnie d’assurance, incluant :

- Diagnostic du périmètre et compréhension des processus
- Cartographie des risques IT, cyber et IA
- Construction du plan de travail
- Tests, analyses de données et contrôles techniques
- Rédaction des constats et recommandations
- Suivi des recommandations et reperforming
- Intégration des nouveaux enjeux liés à l’IA (gouvernance, risques, contrôles)

### Objectifs du projet

1. Développer mes compétences en audit et inspection IT  
2. Approfondir ma compréhension des risques IT, cyber et IA dans l’assurance  
3. Produire un dossier structuré, professionnel et aligné avec les attentes d’un poste d’auditeur IT  
4. Maîtriser GitHub et l’utiliser comme support de documentation et de versioning  

#### Structure du projet

- 01_Context_Diagnostic : contexte, périmètre, diagnostic initial  
- 02_Risk_Assessment_Plan_de_travail : risques IT, cyber, IA, plan de travail  
- 03_Tests_et_Analyses : programme de tests, analyses de données, scripts  
- 04_Rapport_et_Recommandations : constats, recommandations, rapport final  
- 05_Suivi_des_recommandations : plan de suivi, reperforming  
- 06_Annexes : référentiels, normes, documentation
